01 Mar 2018, 04:11
linux / 进程 / 端口

通过进程查询端口

查询获取进程PID

$ ps aux|grep aria2c
isees      2704  1.4  0.1 100772 12912 ?        Rs   03:36   0:39 aria2c -D
isees      3132  0.0  0.0  15964   972 pts/0    S+   04:20   0:00 grep --color=auto aria2c
