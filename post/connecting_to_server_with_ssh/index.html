Connecting to server via ssh

10 Jul 2019, 21:20
ssh / linux / mac

1. Generate a Key Pair

$ ssh-keygen -t rsa -b 4096

2. Upload your Public Key

$ ssh-copy-id -i KEYNAME USERNAME@IP -p PORT

3. Connecting to server via ssh

$ ssh -i KEYNAME -p 'PORT' 'USERNAME@IP'
