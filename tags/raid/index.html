ZFS - CREATE DISK POOLS

07 Aug 2019, 19:46
zfs / raid / ubuntu

This tutorial will cover how to create pools with different RAID levels. singular/basic (no RAID) raidz raidz2 raid 0 raid 10 Creating the pools by specifying disks like /dev/sda and /dev/sdb is somewhat dangerous. Your pool could become degraded or disappear if you plug in another drive and all the letters change. To remedy this, use paths with /dev/disk/by-id/... instead. Also, if your pool does become degraded because of the situation described above, running those export/import commands will fix the situation, even if you have already written data since the pool became degraded.
